## Commands
* `run` : runs the programm
* `test` : tests the programm
* `n1` `n2` `n3` `n4` : new Game with 1-4 players
* `a` `b` `c` `d` : to choose a player 
* `1` `2` `3` `4` : to choose a figure 1-4
* `q` : exits the Game
